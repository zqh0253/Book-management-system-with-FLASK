# DBS's project: Book Management System
Bulit with `python flask`<br>
## How to run?
command:<br>
> venv\Scripts\activate<br>
> (For Windows:) set FLASK_APP = bookdb.py<br>
> (For MacOS:) export FLASK_APP = bookdb.py<br>
> flask run<br>
